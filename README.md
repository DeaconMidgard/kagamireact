# kagami

kagami overlay for kagami plugin.



## Installation

// 사진

1. Setup [kagami plugin]()
2. set URL `https://ramram1048.github.io/kagamireact/`



## Features

- Customize animation window:
  animation speed, icon size, etc.
- Save rotations and Reset rotation lists
